#### **Counter App** *(under development)*

* Basic features: **Increment, Decrement, Reset**.
* Includes a **Settings Screen** with options like:

  * Vibrate on click
  * Play sound on click
  * Use hardware buttons
  * Keep screen on
  * Theme change
  * Export data
